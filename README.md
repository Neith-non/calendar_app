SET UP
Step 1:
download https://getcomposer.org/Composer-Setup.exe

Step 2: Run the Installer
1. Open the Composer-Setup.exe file you just downloaded.

2. Choose Install for all users (it might ask for administrator permission).

3. Leave "Developer mode" unchecked and click Next.

4. ⚠️ CRITICAL STEP: The installer will ask you to choose your command-line PHP. You need to point this to the PHP that comes with XAMPP.

  - Click the Browse button.

  - Navigate to C:\xampp\php\

  - Select the php.exe file.

  - Check the box that says "Add this PHP to your path" (if it asks) and click Next.

5. Leave the proxy settings blank and click Next, then Install.

Step 3: 
Close the terminal (or PowerShell/VS Code terminal) you were just using.
Open a brand new terminal window.

Step 4:
IN TERMINAL
check if installed ya run: composer -v
then run: composer install
